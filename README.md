# eslint-config-gerard

## Install
```bash
yarn add -D eslint-config-gerard
```

## Extend this config in your eslintrc
```json
{
  "extends": ["eslint-config-gerard"]
}
```

## Run
```bash
eslint --ext .js,.jsx . --fix
```
Remember .eslintignore
```
**/node_modules/**
**/build/**
**/dist/**
**/lib/**
````
